# buildings-vs-forest-classification
Бинарная классификация изображений леса и зданий с помощью CNN, TensorFlow/Keras, PyTorch и VGG19.

# Buildings vs Forest Image Classification

Проект посвящён бинарной классификации изображений на два класса: **лес** и **здания**.  
В работе реализованы и сравнены несколько подходов к решению задачи компьютерного зрения: сверточная нейронная сеть на TensorFlow/Keras, собственная CNN-модель на PyTorch и модель на основе transfer learning с предобученной VGG19.

## Цель проекта

Цель работы — построить и сравнить несколько моделей компьютерного зрения для классификации изображений леса и зданий, а также определить, какой подход показывает наилучшее качество на тестовой выборке.

## Данные

В проекте используется датасет изображений, разделённый на обучающую и тестовую выборки.

Классы изображений:

- `0 - forest` — изображения леса
- `1 - buildings` — изображения зданий

Размер датасета:

| Выборка | Количество изображений |
|---|---:|
| Train | 3562 |
| Test | 900 |

Все изображения приводятся к размеру **150 × 150 пикселей**.

## Этапы работы

В ходе проекта были выполнены следующие шаги:

1. Подключение Google Drive и загрузка архива с датасетом.
2. Распаковка изображений в рабочую директорию.
3. Первичный просмотр изображений из обоих классов.
4. Подготовка генераторов данных для обучения и тестирования.
5. Применение аугментации изображений для обучающей выборки.
6. Построение CNN-модели на TensorFlow/Keras.
7. Обучение модели Keras с использованием EarlyStopping.
8. Проверка модели на произвольных изображениях леса и зданий.
9. Построение собственной CNN-модели на PyTorch.
10. Обучение PyTorch-модели и оценка качества на тестовой выборке.
11. Реализация transfer learning с использованием предобученной VGG19.
12. Сравнение результатов всех моделей.

## Аугментация данных

Для увеличения разнообразия обучающих изображений использовались следующие преобразования:

- поворот изображения;
- сдвиг по ширине и высоте;
- масштабирование;
- shear-преобразование;
- горизонтальное отражение;
- нормализация значений пикселей.

Аугментация применялась только к обучающей выборке. Тестовые изображения использовались без случайных преобразований.

## Используемые модели

### 1. CNN на TensorFlow/Keras

Первая модель была реализована с помощью TensorFlow/Keras.

Архитектура модели:

- Input layer размером `150 × 150 × 3`;
- несколько сверточных слоёв `Conv2D`;
- слои `MaxPooling2D`;
- `BatchNormalization`;
- `Flatten`;
- полносвязный слой `Dense`;
- `Dropout`;
- выходной слой с `sigmoid` для бинарной классификации.

Для обучения использовались:

- функция потерь `binary_crossentropy`;
- оптимизатор `Adam`;
- learning rate `0.00005`;
- метрика `accuracy`;
- EarlyStopping по `val_loss`.

Модель обучалась до 50 эпох с восстановлением лучших весов.

### 2. CNN на PyTorch

Вторая модель была реализована на PyTorch.

Архитектура модели включала:

- несколько сверточных слоёв `Conv2d`;
- функцию активации `ReLU`;
- слои `MaxPool2d`;
- `BatchNorm2d`;
- `Flatten`;
- полносвязные слои;
- выходной слой с одним логитом.

Для бинарной классификации использовалась функция потерь:

```python
BCEWithLogitsLoss
```

Для оптимизации применялся:

```python
Adam
```

Модель обучалась на GPU, если он был доступен.

### 3. Transfer Learning с VGG19

Третья модель использовала предобученную архитектуру **VGG19**.

В качестве основы была взята модель VGG19, обученная на ImageNet.  
Сверточная часть VGG19 использовалась как feature extractor, а поверх неё были добавлены собственные полносвязные слои для бинарной классификации.

Обучение проходило в два этапа:

1. **Заморозка сверточной базы VGG19**  
   На первом этапе обучались только новые полносвязные слои.

2. **Fine-tuning**  
   На втором этапе сверточная база была разморожена, а learning rate уменьшен для более аккуратного дообучения модели.

Такой подход позволил получить лучшее качество среди всех рассмотренных моделей.

## Результаты

По итогам экспериментов были получены следующие результаты:

| Модель | Test Accuracy |
|---|---:|
| CNN TensorFlow/Keras | ~0.983 |
| CNN PyTorch | ~0.987 |
| VGG19 Transfer Learning | ~0.998 |

Лучший результат показала модель на основе **VGG19**, использующая transfer learning и последующее fine-tuning.

## Проверка работы моделей

После обучения модели проверялись на тестовой выборке.  
Также для Keras-модели была выполнена проверка на отдельных произвольных изображениях:

- изображение леса;
- изображение здания.

Модель корректно определила классы тестовых изображений, выдавая значение, близкое к `0` для леса и близкое к `1` для здания.

## Вывод

В рамках проекта была решена задача бинарной классификации изображений леса и зданий.

Были реализованы и сравнены три подхода:

- собственная CNN-модель на TensorFlow/Keras;
- собственная CNN-модель на PyTorch;
- модель с transfer learning на основе VGG19.

Все модели показали высокое качество классификации, однако наилучший результат дала модель на базе **VGG19**. Это связано с тем, что VGG19 уже была предварительно обучена на большом наборе изображений ImageNet и умеет извлекать универсальные визуальные признаки.

Таким образом, для данной задачи наиболее эффективным оказался подход transfer learning с последующим fine-tuning.
